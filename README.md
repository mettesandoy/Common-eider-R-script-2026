# Common-eider-R-script-2026

R script for replication of results and figures presented in [Spatial use and movement patterns of common eiders during breeding in southern Norway].
DOI:


Note: This script is not designed to run end-to-end in a single execution. The workflow includes intermediate steps performed in QGIS and manual data collection, which are documented inline using # comments throughout the script. These comments explain what needs to be done between each R section before proceeding.
